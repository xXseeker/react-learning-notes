## Что такое React?

React - это библиотека для создания интерфейсов.

Главная идея React - собирать интерфейс из **компонентов**.

Компонент - маленький кусочек интерфейса, например:

- кнопка;
- шапка сайта;
- карточка поста;
- поле ввода и т.д.

Потом из таких частей собирается вся страница.

## Зачем нужен React?

Без React большие сайты делать сложнее, потому что:

- много повтояющегося кода;
- трудно управлять изменениями на странице;
- интерфейс становится запутанным.

React помогает:

- переиспользовать компоненты;
- удобно обновлять экран;
- писать код более структурно.

## Что такое JSX?

JSX - это специальный синтаксис в React, который выглядит почти как HTML.

Он нужен чтобы удобно было описывать интерфейс прямо в JavaScript.

```javascript
const element = <h1>Привет мир!</h1>;
```

Похоже на HTML, но это не он.

- В JSX можно вставлять JavaScript код через {}

```javascript
function App(){
    const name = "Леша"

    return (
        <div>
            <h1>Привет, {$name}</h1>
        <div>
    )
}
```

- Всегда нужно возвращать лишь один родительский элемент.

Так нельзя:

```javascript
function App(){
    const name = "Леша"

    return (
        <div>
            <h1>Привет, {$name}</h1>
        <div>
        <div>
            <h2>Как дела?</h2>
        <div>
    )
}
```

Правильный вариант(возвращается только один div, в котором уже находятся другие элементы):

```javascript
function App(){
    const name = "Леша"

    return (
        <div>
            <h1>Привет, {$name}</h1>
            <h2>Как дела?</h2>
        <div>
    )
}
```
