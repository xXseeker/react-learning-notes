## Компонент

Компонент - это отдельная часть интерфейса, которую можно написать один раз и использовать много раз.

Если совсем по простому, то компонент - это кусочек страницы.

Например сайт можно разбить на блоки:

- шапка сайта
- меню
- карточка товара
- подвал

Каждая часть может быть компонентом.

Компоненты нужны, чтобы код был понятнее, удобнее и самое важное переиспользуемым.

Компонент - это функция, которая возвращает JSX.

```javascript
function App(){
    return <h1>Привет, React!</h1>
}

export deafault App
```

- App(){...} - имя компонента
- reaturn () возвращает то, что будет показано на странице
- <h1>...</h1> - JSX

## Как используется компонент:

Чтобы использовать компонент его нужно вызвать как HTML-тег в другом компоненте:

```javascript
function Welcome() {
  return <h1>Привет!</h1>;
}

function App() {
  return (
    <div>
      <Welcome />
    </div>
  );
}
```
